## üñ•Ô∏è Installation

### Frontend

1. Clone this repo
```bash
git clone https://github.com/lucas-araujo-costa/frontend-challenge.git
```

2. Go to the project folder
```bash
cd frontend-challenge
cd capputeeno
```

3. Install dependencies
```bash
npm install
```

4. Run Front-end
```bash
npm run dev
```

### Running the API


1. Go to the API folder

```bash
cd frontend-challenge
cd api
```

2. Install dependencies
```bash
yarn install
```

3. Run start
```bash
yarn start
```
